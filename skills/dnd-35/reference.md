# Quick 3.5 reference

## Bonus types (stacking)

**Same-type** bonuses generally **do not stack**; only the highest applies. Bonuses from **different sources** (e.g. enhancement + morale) generally stack. Important exceptions: dodge stacks with dodge; circumstance sometimes does not stack per description.

Abilities: enhancement to an ability **does not** stack with another enhancement to the same ability (usual same-type rule applied to the effect).

## Actions (common in combat)

- **Standard**: single attack (or maneuver that uses a standard action), cast standard-action spell.
- **Move**: equivalent to ~30 ft for Medium Human; may split with standard action in some cases.
- **Free**: speak, drop object; limited by the DM.
- **Immediate**: outside your turn, 1 per round if available.
- **Swift**: only 1 per turn (except spells with swift casting time under their own rules).

## Common formulas

- **Melee attack**: BAB + Str (or Dex if finesse weapon) + size + others.
- **AC**: 10 + armor + shield + Dex (capped) + size + natural + deflection + others.
- **Spell DC**: 10 + spell level + class key ability.

## SRD and sources

SRD content: core classes, core spells, base monsters, base items. Supplements: mark as non-SRD and do not copy full text.
